como recorto una imagen con css

CSS no permite recortar una imagen de forma nativa, pero puedes simular el recorte de una imagen utilizando la propiedad overflow: hidden; en un contenedor que rodea la imagen. Esto ocultará cualquier parte de la imagen que esté fuera del contenedor. Aquí tienes un ejemplo:
